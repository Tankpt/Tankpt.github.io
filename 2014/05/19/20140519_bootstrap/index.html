boostrap的一些资源清单

自己整理了一些boostrap的资源清单，列下顺便整理书签

bootstrap资源收集

之前入门的时候是从bs开始入手的，花过挺多的时间去浏览网站上照一些资源，看到好的都会默默留在书签里，干脆一起整理下分享，好让需要的人可以简单点

##一、botstrap有关插件##

  1. Bootstrap Tour 关于分步介绍的插件
  2. bootstrap-lightbox 关于图片展示的一个插件，点击之后会有一个类似遮罩的效果
  3. Bootstrap video player 视频播放插件
  4. bootstrap-hover-dropdown 一个点击按钮出现下拉框的插件，比较扁平化
  5. DataTables - Blog表格排序
  6. Flippant.js 点击修改文档的，类似bootstrap中自带的那个modal一样，不过效果很炫
  7. bootstrap-datetimepicker时间插件
  8. Bootbox.js 同6，类似modal那个效果
  9. X-editable 提供一个可以修改文档的方法，还不错效果

##二、bootstrap有关资源##

  1. 最全的 Twitter Bootstrap 开发资源清单 之前在网上偶尔看到的很多资源的一个集合
  2. The Best BootStrap Resources 同10资源非常的丰富，根据需要进行筛选
  3. Ridder 一个修改的bootstrap的主题
  4. Bootsnipp bootstrap上一些集成的组件
  5. Intro.js 一个类似1的插件
  6. TreeGrid 树型的插件
  7. jquery-treegrid同15也是一个树形的插件
  8. Bootstrap Image Gallery一个展示图片和视频的插件。效果类似自带的modal
  9. 30个惊人的插件来扩展 Twitter Bootstrap
  10. WordPress Themes, Reviews and News一些相关的主题
  11. 25个强大、实用的Bootstrap工具和生成器
  12. 40 个Bootstrap工具和生成器
  13. Bootstrap Expo一些主题挺不错的
  14. flatstrap同是bs的一些主题，扁平化的多
  15. bootsnipp貌似国外的一个bs资源的网站
  16. jqBootstrapValidationbs的一个验证的插件
  17. GitHub托管BootStrap资源汇总一个人他的一些收集
  18. Fuel UX一个继承了很多新功能的插件的地方，比如树形，时间等，作者貌似定制了一份
  19. 最全的 Twitter Bootstrap 开发资源清单又是某一个最全
  20. 12款最佳的Bootstrap设计工具csnd上的
  21. 一款后台主题超级赞！！看了就知道了不过貌似要付费的
  22. bootstrappage 一些免费的bs主题
  23. 免费的响应式bootstrap管理员后台界面主题
  24. wrapbootstrap一些bs的后台的主题
  25. matrix-admin不错的一个后台主题
  26. portnine-free-bootstrap-theme国人写的一个主题托管在csdn上

  1. bootcss挺赞的
  2. metro-bootstrap
  3. Metro JS
  4. 一些metro风格的介绍
  5. metro-bootstrap
  6. bootmetro
  7. bootswatch看着挺赞的
